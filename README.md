# Enterprise Image Generator

Enterprise Image Generator is an AI-powered web application that generates images from text prompts.  
It uses a FastAPI backend and a frontend interface to interact with the image generation service.

## Features

- Generate AI images from text prompts
- FastAPI backend API
- Frontend user interface
- Hugging Face API integration
- Stores generated output images
- Clean project structure with backend and frontend separation

## Tech Stack

### Backend
- Python
- FastAPI
- Uvicorn
- Hugging Face API

### Frontend
- React
- Vite
- JavaScript
- CSS
